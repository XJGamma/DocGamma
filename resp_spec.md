# 服务端返回数据格式说明

## 格式
```
{
  'code':     int,        // 返回码
  'message':  string,     // 返回消息
  'data':     string,     // Base64编码的二进制数据
  'checksum': string      // 数据校验码
}
```

## 返回码和返回消息
可以根据需要在客户端显示，欢迎修改
### 服务器内部消息
```
0  '服务器错误'
1  '客户端参数错误'
```
### 注册
```
2  '注册成功'
3  '该用户名已注册'
```
### 登录
```
4  '登录成功'
5  '用户名不存在'
6  '密码错误'
```
### 注销
```
7  '注销成功'
```
